# PaintPola

PyQt6 製のデスクトップペイントアプリケーション。
レイヤー・ブレンドモード・エフェクト・アニメーション・ワンクリックアクションに対応。

## 技術スタック

| 区分 | 技術 |
|------|------|
| 言語 | Python 3.13 |
| GUI | PyQt6 (Qt 6.11) |
| 画像処理 | numpy, OpenCV (cv2) |
| GIF出力 | Pillow (PIL) |
| テスト | pytest, pytest-qt |

## 起動方法

```
start.bat
```

または直接:

```
python main.py
```

## 機能一覧

### 描画ツール

| ツール | 説明 |
|--------|------|
| ペン | 丸ペン・ソフト・スプレー・マーカー・ぼかしの5種。サイズ1〜200px |
| 消しゴム | サイズ1〜300px |
| バケツ塗り | フラッドフィル。拡張/縮小 -30〜+30px |
| 直線 / 四角形 / 楕円 | 図形描画。枠線のみ / 塗りのみ / 枠線＋塗り |
| テキスト | キャンバス上にテキスト配置 |
| スポイト | クリックで色を取得。Alt キーで一時スポイト |
| 矩形選択 / 投げなわ | 選択モード / 変形モード切替対応。変形モード・ピボット指定もオプションから設定可能 |
| 囲み内塗りつぶし | 投げなわで囲んだ範囲内にある、線で閉じた領域だけを自動検出して塗りつぶす（クリスタ風）。マウスを離すと自動実行 |
| 移動 | レイヤー移動。矢印キー 1px、Shift+矢印 10px |
| 自由変形 | 選択範囲の移動・拡縮・回転。変形モード・ピボット指定対応 |

対称定規（左右対称描画）対応。

### 変形機能

矩形選択・投げなわ・自由変形ツールのオプションから変形モードを切り替えられる。

| モード | 説明 |
|--------|------|
| 拡縮・回転 | 8方向ハンドルによるリサイズと回転ハンドルによる回転 |
| 自由変形（パース） | 4隅を個別にドラッグして遠近感のある変形。四角形内ドラッグで全体移動。回転ハンドルにも対応 |
| メッシュ変形 | N×N 分割の格子を制御点として自由に歪ませる変形。分割数 2〜8 で調整可能。回転ハンドルにも対応 |

選択範囲なしでレイヤー全体を変形したい場合は、画像メニューの「レイヤーを変形（拡大縮小・回転）...」または変形メニューの「拡大縮小・回転（%指定）...」を使う。スライダーで % を動かすとリアルタイムでプレビューされ、OK で確定・キャンセルで元に戻る。

#### 変形ピボット（基準点）

拡縮・回転時に基準点を指定できる。

- **プリセット（9点）** — アンカーウィジェットで四隅・辺中点・中央から選択
- **任意（ドラッグ）** — キャンバス上の青い十字マーカーをドラッグで自由配置

### レイヤー

- 追加・削除・複製・並べ替え（上下移動）
- グループ（フォルダ）で一括管理。レイヤーのグループ追加・取り出しボタン対応
- ドラッグ＆ドロップによるレイヤー順序変更・グループへの移動
- 不透明度 (0〜100%)・クリッピング・参照レイヤー・表示/非表示
- ブレンドモード: 通常 / 乗算 / スクリーン / オーバーレイ / 加算
- 統合: 下に統合 / 全表示統合 / フォルダ統合（すべてUndo対応）
- アイコンベースのコンパクトUI（クリスタ風）

### レイヤーエフェクト

| エフェクト | パラメータ |
|------------|------------|
| 縁取り | サイズ、色 |
| ドロップシャドウ | 強度、X/Yオフセット、ぼかし、色 |
| グロー（発光） | 強度、サイズ、色 |
| ガウスぼかし | 強度、半径 |
| HSL調整 | 色相・彩度・明度 |

合成順: 縁取り → シャドウ → グロー → ぼかし → HSL

**ラスタライズ**（画像メニュー → レイヤーをラスタライズ）— 設定中の効果を画像ピクセルに焼き込み、エフェクト設定を無効化する。統合（下に統合等）でも同様に効果が自動で焼き込まれるため、統合後に効果が消えたり二重適用されたりしない。

### アクション

レイヤーパネルの「アクション」タブからワンクリックで実行。すべてUndo対応。

| アクション | 内容 |
|------------|------|
| 線画ずらし（色収差） | 赤・青・黄の色ずれを生成。移動量・回転・拡縮を設定可能 |
| グロー / 発光 | 暗い背景＋色付きグローを自動セットアップ |
| 影付け | ドロップシャドウレイヤーを自動生成 |
| 背景パターン生成 | ドット・ストライプ・チェック・グラデーションなど8種 |
| 線画色変え | 線画の色を一括変換。プリセット付き |
| ポップアウト | 白縁＋影でステッカー風に浮き出し |
| ランダムタイリング配置 | 個数・スケール・回転・重なりを指定し、レイヤーをランダムに複製配置。壁紙パターン作成等に便利。統合 / グループ出力を選択可 |
| パスに沿った連続複製 | キャンバスをクリックしてパスを描き（Enter で確定 / Backspace で1点削除 / Esc でキャンセル）、軌跡に沿って等間隔で複製配置。進行方向への自動回転に対応 |
| 紙質感グレイン | ノイズ粒子を重ねて紙のようなザラついた質感を付与。強度・粒の粗さ・合成モード（オーバーレイ / 乗算）を指定可能 |

### キャンバス

- デフォルトサイズ: 2500 x 2500px
- ズーム: マウスホイール（5%〜500%）
- パン: 中ボタンドラッグ / Space+ドラッグ
- 回転: ナビゲーターの回転ボタン（左右回転・リセット）
- 左右反転表示
- サイズ変更: 9点アンカー + リアルタイムプレビュー
- 透明チェッカーボード: 背景がない部分にグレー×白の格子を表示

### ナビゲーター

- キャンバス全体の縮小プレビュー

### アニメーション（パラパラ漫画）

### 内部構造

**ブラシエンジン** — `BrushBase` 継承の5種（丸ペン / ソフト / スプレー / マーカー / ぼかし）。Canvas は `stroke_to()` だけを呼ぶ。

**エフェクト処理** — `Layer.image_with_effects()` で numpy/cv2 を使い合成。縁取り(膨張) → シャドウ(ぼかし+オフセット) → グロー(膨張+ぼかし+着色) → ブラー → HSL。

**変形エンジン** — 3つのモードを統一的に管理:
- 拡縮・回転: QTransform ベースの行列変換
- 自由変形（パース）: `cv2.getPerspectiveTransform` で4隅の射影変換
- メッシュ変形: N×N 分割した格子の各セルに `cv2.warpPerspective` を適用して合成

**履歴システム** — タグ付きタプルで2種類を管理:
- `("pixel", layer_id, QImage)` — 描画前の画像コピー
- `("structure", snapshot)` — 全レイヤーの再帰的スナップショット

**.pola 形式** — ZIP内に `meta.json`（レイヤー構成・エフェクト設定）+ 各レイヤーPNG。読み込み時にサイズ・パラメータ・パスのバリデーションを実施。
